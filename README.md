## SYSTEM OVERVIEW

A fully immersive developer portfolio featuring a real-time 3D mech warrior, scroll-driven camera animation, and a HUD-inspired UI aesthetic. Built with performance and visual impact in mind.

---

## TECH STACK

| Layer | Technology |
|---|---|
| Frontend | React 18 + Vite 5.4 |
| 3D Engine | Three.js 0.160 + React Three Fiber |
| Animation | GSAP + ScrollTrigger + Framer Motion |
| Smooth Scroll | Lenis |
| Styling | Tailwind CSS v3 |
| Contact | EmailJS |
| Deploy | Vercel |

---

## FEATURES

- ⚡ Real-time 3D mech warrior with bloom glow & neon lighting
- 🎬 Scroll-driven camera — 5 distinct phases across sections
- 🖱️ Custom HUD cursor with cyan ring
- 🔊 Ambient sound toggle via Web Audio API
- 📱 Fully responsive — mobile to 4K
- 🚀 GLB compressed 44MB → 4.2MB (Draco + WebP textures)
- 🎯 Loading screen with HUD boot sequence
- 📊 Scroll progress bar
